This file contains the instructions to run this project
There are two ways to run this project

1. Import the project in eclipse, here are the steps
- Select import - Projects from SVN
- Enter repository path https://svn.riouxsvn.com/fevo/Feevo/
- Run class WorldGUI

2. As jar file
-Download files feevo.jar,background.jpg,fish2.jpg,worm.jpg,chromosomes.txt in same folder
-Run jar with command java -jar feevo.jar

To run the project
For fitness function hungry or natural
-Select either of the fitness functions and click evolve

For fitness function manual 
-Click on any fish
-Enter a number from 1 to 100 in the text box near above score button
-Click on score
- Do this for a few fish
- Click evolve
